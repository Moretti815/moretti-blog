# VitePress 文章模糊标签效果展示

本站使用自定义 markdown-it 插件，支持通过 `:blur[文字]` 语法为文本添加模糊效果，鼠标悬停时文字变清晰。

## 语法说明

使用 `:blur[]` 包裹需要模糊的文字：

```md
:blur[你想要模糊的文字]
```

## 基础用法

这是一段普通文字，其中包含 :blur[你知道得太多了。] 这样的模糊文字。

## 行内使用

模糊标签可以在段落中 :blur[随意穿插使用]，不影响其他文字的显示效果。

也可以在 **粗体文字后面** 紧跟 :blur[隐藏的内容]，或者在 `代码` 旁边使用 :blur[剧透警告]。

## 嵌套 Markdown

模糊标签内部支持嵌套 Markdown 语法：

- :blur[**加粗的模糊文字**]
- :blur[*斜体的模糊文字*]
- :blur[`代码样式的模糊文字`]

## 使用场景

1. **剧透隐藏**：最新一集的结局是 :blur[主角其实早就知道真相了]
2. **答案隐藏**：1+1 的答案是 :blur[2]
3. **敏感信息**：联系电话 :blur[138-0000-0000]
4. **彩蛋内容**：彩蛋在 :blur[最后一页的左下角]

## 效果特点

- 默认状态文字完全模糊，无法辨认
- 鼠标悬停时平滑过渡变清晰
- 移开鼠标后重新变模糊
- 支持行内使用，不影响段落排版
